PROJECT OVERVIEW :- This project focuses on analyzing historical sales data to understand monthly sales trends. The analysis is performed using Python, primarily leveraging Pandas for data manipulation and Matplotlib for data visualization. The complete analysis is documented in a Jupyter Notebook and version-controlled using Github.
OBJECTIVES :- 
* Load and explore raw sales data from a CSV file.
* Clean and preprocesses the dataset
* Calculate total daily and monthly sales
* Analyze sales trend over time
* Visualize montly sales using bar charts
* Save plots programmatically for reporting

DATASET DESCRIPTION:- The dataset contains daily sales information with the following key columns:
* Date - Date of sales record
* Q-P1,Q-P2,Q-P3,Q-P4 - Quantity sold for different products
* Sales price columns for corresponding products
NOTE:- An extra index column (Unnamed) generated during CSV export was removed during data cleaning.

TOOLS & TECHNOLOGIES USED:-
* Python
* Pandas
* Matplotlib
* Jupyter Notebook
* Git and Github
